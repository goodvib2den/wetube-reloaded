## URL List

### Global route

- / -> home
- /join -> Join
- /login -> Login
- /search -> Search

### User route

- /users/:id -> View profile
- /user/logout -> Log Out
- /users/edit -> Edit user profile
- /users/delete -> Delete user profile

### Video route

- /videos/:id -> Watch video
- /videos/:id/edit -> Edit video
- /videos/:id/delete -> Delete video
- /videos/upload -> Upload Videoo
